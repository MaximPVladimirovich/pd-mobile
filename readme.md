# pd-mobile

## Setting up local environment

### Pre-requisites
- npm >= 7.10
- node >= 15
- ts >= 3.8
- cocoapods 1.10.1

### Compiling
- Install all node_module dependencies via `npm install`
- Generate pod file: 
  -- `cd ios`
  -- `pod install --repo-update`
- Go back to root folder and run either `yarn ios` / `yarn android`
